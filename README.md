World Countries Directory (Довідник країн світу)
Веб-додаток для вивчення інформації про країни світу, їх порівняння та зручного пошуку. Проєкт використовує реальні дані з REST Countries API.

Основні можливості
Інтерактивний пошук: Швидкий пошук країни за її офіційною англійською назвою.

Розумні підказки (Autocomplete): Під час введення назви програма пропонує варіанти країн, що значно пришвидшує роботу та запобігає помилкам у написанні.

Детальна інформація: Перегляд прапора, столиці, населення, регіону, валюти та списку країн-сусідів.

Порівняння країн: Можливість додати до 3-х країн у спеціальну зону для одночасного порівняння їхніх характеристик.

Фільтрація та сортування: Гнучке відсіювання за регіонами (Азія, Європа тощо) та сортування за кількістю населення.

Адаптивний дизайн: Коректне відображення прапорів зі збереженням їхніх природних пропорцій (без сплющування).

🛠 Технології
HTML5 — структура сторінки.

CSS3 — стилізація, включаючи Flexbox, Grid та властивість object-fit для медіа.

JavaScript (ES6+) — логіка роботи з API, асинхронні запити (fetch), динамічне оновлення DOM та система автодоповнення.

API — REST Countries V3.1.

📦 Як запустити проєкт
Склонуйте або завантажте репозиторій.

Переконайтеся, що всі файли (index.html, style.css, script.js) знаходяться в одній папці.

Важливо: Відкрийте проєкт через локальний сервер. У VS Code рекомендується використовувати розширення Live Server.

Чому? Браузери блокують прямі запити до API, якщо файл відкритий просто подвійним кліком (file://).

Введіть назву країни англійською мовою в полі пошуку (наприклад, Ukraine, Germany або Kazakhstan).

📋 Особливості реалізації
Виправлення пропорцій: Завдяки CSS властивості object-fit: contain, прапори країн з нестандартними розмірами (як у Казахстану чи Катару) більше не виглядають сплюснутими.

Оптимізація API: Додаток використовує фільтрацію полів (?fields=...), що пришвидшує завантаження даних та запобігає помилкам сервера (Bad Request 400).

UX підказки: Поля вводу містять плейсхолдери, які орієнтують користувача щодо мови введення.
